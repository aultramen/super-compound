# Rule Index

Compact startup rule:

- Use evidence before claims.
- Route through current `/sc-*` workflows only.
- Load contracts first, then full files on demand.
- Preserve BRD -> PRD -> FSD -> GOAL authority.
- Stop on `OPEN-*` blockers.
- Keep issue files as pointers, not copied prose.
- Verify before completion.
- Route Git operations through `/sc-go` with preview-first safety.
