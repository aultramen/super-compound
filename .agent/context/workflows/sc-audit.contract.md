# /sc-audit Runtime Contract

Purpose: read-only security, compatibility, compliance, agent-surface, and release-readiness audit.

Scope the risk, inspect manifests/config/hooks/prompts/env examples, run safe checks, and report confirmed findings, risks, and assumptions by severity. For release readiness, inspect Git state read-only for direct-main risk, dirty tree, unpushed branch, and secret-looking files. Stop before applying changes.
